Proyecto Donaciones App (Angular + TypeScript)

Incluye 5 módulos CRUD:
1. Campañas
2. Voluntarios
3. Puntos de recolección
4. Tipos de donación
5. Beneficiarios

Pasos para ejecutarlo:

1. Asegúrate de tener Node.js y Angular CLI instalados.
2. Abre una terminal en la carpeta del proyecto 'donaciones-app'.
3. Ejecuta:
   npm install
4. Luego:
   ng serve
5. Abre en el navegador:
   http://localhost:4200

La carpeta 'public/json' contiene los archivos JSON usados como "base de datos".
